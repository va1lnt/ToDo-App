# ToDo App

Просто приложение ToDo, написанное на Python 3 и скомпилированное в приложение под Windows.

## Описание

Это простое приложение для управления задачами. Оно позволяет пользователям создавать, редактировать и удалять задачи, а также отслеживать их выполнение.

![Скриншот приложения](https://github.com/va1lnt/ToDo-App/raw/main/screenshot.png)

## Установка

1. Скачайте последнюю версию приложения с [GitHub Releases](https://github.com/va1lnt/ToDo-App/releases).
2. Запустите скачанный файл.

## Использование

1. Откройте приложение.
2. Введите новую задачу в поле "Введите новую задачу..." и нажмите "Добавить".
3. Чтобы удалить задачу, нажмите на кнопку "x" рядом с задачей.
4. Чтобы удалить все задачи, нажмите кнопку "Удалить все".

## Технологии

- Python 3
- PyInstaller (для компиляции в исполняемый файл)
